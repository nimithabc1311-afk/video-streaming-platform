# ▶️ StreamZone — Video Streaming Platform

A fully functional **Video Streaming Platform** built using **HTML, CSS, and JavaScript**.

**Developed by: Nimitha BC**

---

## 📌 Features

- 🎬 **Browse Videos** — Grid layout with thumbnails, views, likes and duration
- 🔍 **Live Search** — Search videos by title, channel, category or description
- 🏷️ **Categories** — Filter by Music, Sports, Education, Gaming, Comedy, Technology
- ▶️ **Video Player** — Plays YouTube videos with full controls via iframe embed
- ❤️ **Like Videos** — Like/unlike videos with count update
- 🔖 **Save Videos** — Save and manage your favourite videos
- ⬆️ **Upload Videos** — Upload form with YouTube ID, category, duration and description
- 🔥 **Trending & New** — Home page shows trending and newest videos
- 🔧 **Admin Dashboard** — View all videos with stats, play or delete them
- 💾 **Data Persistence** — All data saved using localStorage
- 📱 **Responsive Design** — Works on mobile and desktop

---

## 🚀 How to Run

1. Download or clone this repository
2. Open `index.html` in any browser
3. No installation needed — videos load from YouTube!
